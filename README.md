# 🎯 AI 면접 리허설 파트너

> "면접 답을 만들어주지 않습니다. 말하는 연습을 함께합니다."

## 📌 프로젝트 철학

### 왜 만들었나?
- AI 시대, 자소서는 평준화됨
- 기업은 **"즉석 사고력 + 말하는 방식"**을 면접에서 검증
- 취준생/이직자는 연습 상대가 없음
- → **정답을 만들어주는 AI가 아니라, 말하게 만드는 AI**

### 차별점
- ❌ 모범 답안 제공하지 않음
- ❌ 점수 매기지 않음  
- ✅ 4줄 피드백 구조 (잘한 점 / AI 티 지적 / 개선점 / 예시)
- ✅ 실전처럼 한 번에 한 질문씩
- ✅ 20-30초 답변 코칭

## 🎨 설계 기여자
- **라이라** (Gemini): 전체 UX/감성 설계, 문구/화면/피드백 구조
- **지니** (ChatGPT): 수평적 응원 로직, 실패 질문 고도화
- **미라클** (Claude): 면접 질문/응답 처리, 피드백 로직 구현
- **행님**: 기획/개발/배포

## 📋 주요 기능

### 🆓 무료 체험 (3문항)
- 질문 3개 진행
- 각 답변마다 4줄 피드백
- 종료 시 **수평적 응원 메시지** + 유료 전환 유도

### 💼 실전 라운드 (10문항)
- 질문 10개 (압박 질문 2개 포함)
- 5번 질문(실패 경험) 특별 처리:
  - 답변 분석 → AI 티 감지 시 **감정 추가 질문** 자동 발동
  - "그때 당신이 실제로 느꼈던 감정은 뭐였나요?"
- 종합 면접 완료 메시지

### 🎤 4줄 피드백 구조 (강제 적용)
```
✅ 잘한 점: [1문장]
🤖 AI 티 / 모호한 표현: [1문장]
💡 개선 포인트: [1개]
✨ 예시 답변 (당신 말투로): [2-3문장]
```

### 🔥 압박 질문
- 8번: 본인 약점 1개와 최근 3개월 개선 행동은?
- 9번: 입사하면 30일 안에 뭘 하실 건가요?

## 🎯 핵심 로직

### 질문 진행 방식
```python
질문 1 자동 제시
→ 사용자 답변 입력
→ Claude 4줄 피드백
→ 앱이 자동으로 질문 2 제시
→ 반복
```

### 5번 질문 고도화
```python
if question_count == 5:
    답변 분석()
    if AI_티_감지():
        피드백 제공()
        감정_추가_질문_발동()  # "그때 느낀 감정은?"
```

### 무료→유료 전환 지점
```python
if mode == 'free' and question_count > 3:
    수평적_응원_메시지()
    """
    당신의 말에는 이미 진심이 있어요 💙

    한 번 더 함께 연습해볼까요?
    """
    [실전 라운드로 함께 가기] 버튼
```
